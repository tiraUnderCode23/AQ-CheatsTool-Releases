NBT EVO Flashing Tutorial with E-Sys + ICOM/ENET

Complete DIY guide for flashing NBT EVO head units

Warning: This procedure involves risks. Make sure you have proper battery charger connected (at least 50A) and backup all data before proceeding.

Launch AQ///Cheat Tool Download E-Sys 3.30.1 PSdZData Full

Introduction

This tutorial will guide you through the process of flashing your NBT EVO head unit to enable features like Apple CarPlay full screen. The process can be done with either ICOM or ENET interface.

ICOM Configuration

For ICOM interface, use these settings:

ConnectionMode = STATIC_IP
ip = 192.168.68.85
Netmask = 255.255.255.0
Gateway = 192.168.68.99

DHCP Server Configuration

Example DHCPSRV.INI configuration:

[GLOBALS]
IPPOOL_1=192.168.68.1-254
IPBIND_1=192.168.68.99
AssociateBindsToPools=1
LEASETIME=86400
NODETYPE=8
SUBNET_MASK=255.255.255.0
ROUTER=192.168.68.1

ENET Configuration

For ENET interface, follow these steps:

  1. Go to Network Connections
  2. Open Properties of your Ethernet adapter
  3. Select Internet Protocol Version 4 (TCP/IPv4)
  4. Configure with these settings: 
    IP address: 169.254.92.38
Subnet mask: 255.255.255.0
  5. Run ZGW_SEARCH from E:/EDIABAS/Hardware/ENET

Step-by-Step Flashing Guide

Step 1: Prepare Your Equipment

  • 50A battery charger connected to OBD port
  • E-Sys 3.30.1 with PSdZData Full
  • ICOM or ENET interface
  • AQ///Cheat Tool (optional but recommended)
Proper setup with battery charger

Step 2: Connect to Vehicle

  1. Launch E-Sys
  2. Click "Open Connection"
  3. Select your connection type (ICOM or ENET)
  4. For ICOM: Use tcp://169.254.92.38:50160
  5. For ENET: Use tcp://169.254.99.82:6801
  6. Click "Connect"
E-Sys connection settings

Step 3: Read Vehicle Data

  1. Read FA (Vehicle Order) and save it
  2. Right-click on FA folder and select "Activate FA"
  3. Click "Read SVT" (Skip BDC_BODY for F85)
  4. Save SVT as SVT_ist
  5. Select same I-Step (Shipment) under Target/KIS/SVT
  6. Select latest I-Step (Target)
  7. Choose "Complete Flash" on right side
  8. Click "Calculate"
  9. Save as SVT_soll
  10. Click "Calculate" under TAL list
  11. Click OK (you may get a warning)
  12. Save calculation as SVT_tal

Step 4: Process TAL

  1. Click "Expert Mode" on left side
  2. Click "Tal..." (3 dots) and open SVT_tal
  3. TAL will show all ECUs needing update
  4. Click "SVT..." (3 dots) and open SVT_soll
  5. Click "FA..." (3 dots) and read FA
  6. Click "Read VIN" outside FA
  7. Select all necessary updates or individual modules
  8. For NBT EVO, select HU_NBT_63 with bFlash, swDeploy, cdDeploy, and ibaDeploy
  9. Click "Check software availability"

Important: Make sure you have all required software files available before proceeding.

Step 5: Start Flashing

  1. Click "Start" to begin flashing process
  2. Monitor progress in ECU Parameters Log
  3. Do not interrupt the process
  4. Wait for confirmation message
Flashing progress in E-Sys

Step 6: Post-Flash Configuration

After successful flash, enable Apple CarPlay full screen:

HMI_ID_VERSION => id6_light
APPLE_ENHANCEMENTS => aktiv
HMI/CARPLAY_FULLSCREEN => aktiv

Success! Your NBT EVO should now be updated to the latest version with full screen Apple CarPlay capability.

Troubleshooting

Error Solution
VCM not activating update [C197] Check battery voltage and connection
TAL validation failed [C205] Recalculate TAL and verify FA
Asynchronous programming [C049] Check all ECUs in TAL and retry
DVD drive not ready Remove any discs before flashing

DANGER ZONE

Improper flashing can brick your head unit. Always:

  • Use proper power supply (50A minimum)
  • Verify all files before flashing
  • Don't interrupt the process
  • Have backup recovery plan
